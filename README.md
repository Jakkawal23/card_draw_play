# Card Draw Play เกมสุ่มการ์ด
เราสามารถกำหนด รูปแบบการ์ด จำนวนการ์ด ที่เราต้องการจะให้สุ่ม เพื่อเล่นเกมได้

## Features
* กำหนดการ์ดได้
* กำหนดจำนวนการ์ดได้
* กำหนด icon สี ของการ์ดได้
* กำหนดรูปแบบการสุ่มได้ (นำกลับไปสุ่มใหม่)
